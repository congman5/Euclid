  PROPOSITION I.6 — Converse of I.5
================================================================================

  Statement: If in a triangle two angles are equal, then the sides opposite the equal angles are also equal.

  System E:
    Given:     ∠abc = ∠acb, ¬(a = b), ¬(a = c), ¬(b = c)
    Prove:     ab = ac

  System H:
    Given:     ¬(ColH(a, b, c)), CongaH(a, b, c, a, c, b)
    Prove:     CongH(a, b, a, c)

  Dependencies: I.3, I.4

  Verified Proof (37 steps) [✓]:
  ──────────────────────────────
   1. [Given]             → ∠abc = ∠acb
   2. [Given]             → ¬(a = b)
   3. [Given]             → ¬(a = c)
   4. [Given]             → ¬(b = c)
   5. [Given]             → on(a, L)
   6. [Given]             → on(b, L)
   7. [Given]             → ¬(on(c, L))
  Case 1 — Reductio: assume ac < ab
     8. [Assume]          → ac < ab
     9. [Prop.I.3]        → between(b, d, a), bd = ac
    10. [let-line]        → on(b, M), on(c, M)
    11. [Angle transfer 9] → ∠dbc = ∠abc        (DA4)
    12. [CN1 — Transitivity] → ∠dbc = ∠acb
    13. [M3 — Symmetry]   → bc = cb
    14. [SAS]             → dc = ab, ∠bdc = ∠cab, ∠bcd = ∠cba, △dbc = △acb
    15. [M8 — Area symmetry] → △dbc = △abc
    16. [Area transfer 4] → (△bdc + △cda) = △bca  (DAr2)
    17. [Area transfer 1] → ¬(△dac = 0)           (DAr1a)
    18. [CN5 — Whole > Part] → △bdc < △bca
    19. [M8 — Area symmetry] → △dbc < △abc
    20. [Contradiction]   → ⊥
  21. [Reductio]          → ¬(ac < ab)
  Case 2 — Reductio: assume ab < ac
    22. [Assume]          → ab < ac
    23. [let-line]        → on(c, N), on(a, N)
    24. [Prop.I.3]        → between(c, e, a), ce = ab
    25. [let-line]        → on(c, P), on(b, P)
    26. [Angle transfer 9] → ∠ecb = ∠acb        (DA4)
    27. [CN1 — Transitivity] → ∠ecb = ∠abc
    28. [M3 — Symmetry]   → cb = bc
    29. [SAS]             → eb = ac, ∠ceb = ∠bac, ∠cbe = ∠bca, △ecb = △abc
    30. [M8 — Area symmetry] → △ecb = △bca
    31. [Area transfer 4] → (△ceb + △bea) = △cba  (DAr2)
    32. [Area transfer 1] → ¬(△aeb = 0)           (DAr1a)
    33. [CN5 — Whole > Part] → △ceb < △cba
    34. [M8 — Area symmetry] → △ecb < △bca
    35. [Contradiction]   → ⊥
  36. [Reductio]          → ¬(ab < ac)
  Trichotomy:
  37. [< trichotomy]      → ab = ac

  Q.E.D. ■

================================================================================

  System E:

    Prove:     d = a

  System H:

    Prove:     d = a

  Dependencies: I.5

  Verified Proof (27 steps) [✓]:
  ──────────────────────────────
   1. [Given]             → on(b, L)
   2. [Given]             → on(c, L)
   3. [Given]             → ¬(b = c)
   4. [Given]             → same-side(a, d, L)
   5. [Given]             → bd = ba
   6. [Given]             → cd = ca
  Reductio: assume ¬(d = a)
     7. [Assume]          → ¬(d = a)
     8. [let-line]        → on(a, R), on(b, R)
  Case 1 — on(d, R):
       9. [Assume]        → on(d, R)
  Case 1a — between(a, d, b):
        10. [Assume]      → between(a, d, b)
        11. [Segment transfer 1] → (ad + db) = ab  (DS1)
        12. [CN3 — Subtraction]  → ad = 0
        13. [M1 — Zero segment]  → d = a
  Case 1b — ¬between(a, d, b):
          14. [Assume]    → ¬(between(a, d, b))

            15. [Assume]  → between(b, a, d)
            16. [Segment transfer 1] → (ba + ad) = bd  (DS1)
            17. [CN3 — Subtraction]  → ad = 0

          24. [Reit]      → d = a                    (from Case 1 conclusion)
      25. [Cases]         → d = a
      26. [Contradiction] → ⊥
  27. [Reductio]          → d = a
