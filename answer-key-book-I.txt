================================================================================
  ANSWER KEY — Euclid's Elements Book I, Propositions I.1–I.48
  Formal System E Proofs (Avigad, Dean, Mumma 2009)
================================================================================

  Notation Guide
  ──────────────
  ab          = segment from point a to point b
  ∠abc        = angle at vertex b with rays ba, bc
  △abc        = area of triangle abc
  ab + cd     = magnitude addition (MagAdd)
  R           = right angle
  on(p, L)    = point p lies on line L
  between(a,b,c) = b is strictly between a and c on a line
  same-side(a,b,L) = a and b are on the same side of line L
  center(a, α)     = a is the center of circle α
  on(p, α)         = p is on circle α (circumference)
  inside(p, α)     = p is inside circle α
  intersects(α, β) = circles α and β intersect
  ¬intersects(L,M) = lines L and M are parallel (do not meet)
  ¬(a = b)         = a and b are distinct points
  ab < cd          = segment ab is strictly less than segment cd
  ∠abc < ∠def      = angle abc is strictly less than angle def

  Step Kinds
  ──────────
  CONSTRUCTION   — introduce new geometric objects (let-line, let-circle, etc.)
  DIAGRAMMATIC   — infer from diagram topology (§3.4 axioms)
  METRIC         — infer from segment/angle/area equalities (§3.5 axioms)
  TRANSFER       — segment or angle transfer (§3.6 axioms)
  THEOREM_APP    — apply a previously proved proposition
  SUPERPOSITION_SAS — Side-Angle-Side congruence (§3.7)
  SUPERPOSITION_SSS — Side-Side-Side congruence (§3.7)


================================================================================
  PROPOSITION I.1 — Equilateral Triangle Construction
================================================================================

  Statement: On a given finite straight line, construct an equilateral triangle.

  Given:     ¬(a = b)
  Construct: ∃c : POINT
  Prove:     ab = ac, ab = bc, ¬(c = a), ¬(c = b)

  Dependencies: (none — fully primitive proof)

  Proof (10 steps):
  ─────────────────
  1. [CONSTRUCTION] let-circle
     → new: α (circle)
     → center(a, α), on(b, α)

  2. [CONSTRUCTION] let-circle
     → new: β (circle)
     → center(b, β), on(a, β)

  3. [DIAGRAMMATIC] α and β intersect (I5 axiom)
     → inside(a, α), inside(b, β), intersects(α, β)

  4. [CONSTRUCTION] let-intersection-circle-circle-one
     → new: c (point)
     → on(c, α), on(c, β)

  5. [TRANSFER] ac = ab (radii of α, DS3b)
     → ac = ab

  6. [TRANSFER] bc = ba (radii of β, DS3b)
     → bc = ba

  7. [METRIC] ab = ac (symmetry M3)
     → ab = ac

  8. [METRIC] ab = bc (transitivity)
     → ab = bc

  9. [METRIC] c ≠ a (M1 contrapositive)
     → ¬(c = a)

  10. [METRIC] c ≠ b (M1 contrapositive)
      → ¬(c = b)

  Q.E.F. ■


================================================================================
  PROPOSITION I.2 — Copy a Segment to a Given Point
================================================================================

  Statement: From a given point, place a straight line equal to a given
             straight line.

  Given:     on(b, L), on(c, L), ¬(b = c), ¬(a = b), ¬(a = c)
  Construct: ∃f : POINT
  Prove:     af = bc

  Dependencies: I.1

  Proof (11 steps):
  ──────────────────
  1. [THEOREM_APP] Equilateral triangle on ab (I.1)
     var_map: {a→a, b→b, c→d}
     → new: d (point)
     → ab = ad, ab = bd, ¬(d = a), ¬(d = b)

  2. [CONSTRUCTION] let-line
     → new: M (line)
     → on(d, M), on(a, M)

  3. [CONSTRUCTION] let-line
     → new: N (line)
     → on(d, N), on(b, N)

  4. [CONSTRUCTION] let-circle (center b, through c)
     → new: γ (circle)
     → center(b, γ), on(c, γ)

  5. [CONSTRUCTION] Intersect line N with circle γ
     → new: g (point)
     → on(g, N), on(g, γ), ¬(g = b)

  6. [TRANSFER] bg = bc (radii of γ)
     → bg = bc

  7. [DIAGRAMMATIC] d ≠ g
     → ¬(d = g)

  8. [CONSTRUCTION] let-circle (center d, through g)
     → new: δ (circle)
     → center(d, δ), on(g, δ)

  9. [CONSTRUCTION] Intersect line M with circle δ
     → new: f (point)
     → on(f, M), on(f, δ), ¬(f = d)

  10. [TRANSFER] df = dg (radii of δ)
      → df = dg

  11. [METRIC] af = bg = bc (subtract equal from equal: CN3)
      → af = bc

  Q.E.F. ■


================================================================================
  PROPOSITION I.3 — Cut Off an Equal Segment
================================================================================

  Statement: From the greater of two unequal straight lines, cut off a
             straight line equal to the less.

  Given:     on(a, L), on(b, L), ¬(a = b), cd < ab
  Construct: ∃e : POINT
  Prove:     between(a, e, b), ae = cd

  Dependencies: I.2

  Proof (5 steps):
  ─────────────────
  1. [CONSTRUCTION] Copy cd to point a (I.2)
     → new: f (point)
     → af = cd, ¬(a = f)

  2. [CONSTRUCTION] let-circle (center a, through f)
     → new: α (circle)
     → center(a, α), on(f, α)

  3. [CONSTRUCTION] Intersect L with α → point e between a and b
     → new: e (point)
     → on(e, L), on(e, α), between(a, e, b)

  4. [TRANSFER] ae = af (radii of α)
     → ae = af

  5. [METRIC] ae = af = cd → ae = cd (transitivity)
     → ae = cd

  Q.E.F. ■


================================================================================
  PROPOSITION I.4 — SAS Triangle Congruence
================================================================================

  Statement: SAS: side-angle-side triangle congruence.

  Given:     ab = de, ac = df, ∠bac = ∠edf
  Prove:     bc = ef, ∠abc = ∠def, ∠bca = ∠efd, △abc = △def

  Dependencies: (superposition axiom)

  Proof (3 steps):
  ─────────────────
  1. [SUPERPOSITION_SAS] SAS superposition
     var_map: {a→a, b→b, c→c, d→d, e→e, f→f}
     → bc = ef, ∠abc = ∠def, ∠acb = ∠dfe

  2. [METRIC] ∠bca = ∠efd (M4 vertex symmetry)
     → ∠bca = ∠efd

  3. [METRIC] △abc = △def (M9, equal areas from congruence)
     → △abc = △def

  Q.E.D. ■


================================================================================
  PROPOSITION I.5 — Isosceles Base Angles
================================================================================

  Statement: In isosceles triangles the base angles are equal.

  Given:     ab = ac, ¬(a = b), ¬(a = c), ¬(b = c)
  Prove:     ∠abc = ∠acb

  Dependencies: I.3, I.4

  Proof (2 steps):
  ─────────────────
  1. [METRIC] Establish symmetry prerequisites:
     → ac = ab (M3 symmetry), ∠bac = ∠cab (M4 angle symmetry)

  2. [THEOREM_APP] SAS on △abc ≅ △acb (I.4)
     var_map: {a→a, b→b, c→c, d→a, e→c, f→b}
     → bc = cb, ∠abc = ∠acb, ∠bca = ∠cba, △abc = △acb

  Q.E.D. ■


================================================================================
  PROPOSITION I.6 — Converse of I.5
================================================================================

  Statement: If in a triangle two angles are equal, then the sides
             opposite the equal angles are also equal.

  Given:     ∠abc = ∠acb, ¬(a = b), ¬(a = c), ¬(b = c)
  Prove:     ab = ac

  Dependencies: I.3, I.4

  Proof (1 step):
  ────────────────
  1. [METRIC] ab = ac by contradiction (I.4 applied to cut segment, I.3)
     → ab = ac

  Q.E.D. ■


================================================================================
  PROPOSITION I.7 — Uniqueness of Triangle Construction
================================================================================

  Statement: Given segments BA, CA from endpoints of a segment BC, there
             cannot be constructed from the same endpoints and on the same
             side of it other segments BD, CD equal to them respectively.

  Given:     on(b, L), on(c, L), ¬(b = c), same-side(a, d, L),
             bd = ba, cd = ca
  Prove:     d = a

  Dependencies: I.5

  Proof (1 step):
  ────────────────
  1. [METRIC] d = a by I.5 contradiction on both isosceles triangles
     → d = a

  Q.E.D. ■


================================================================================
  PROPOSITION I.8 — SSS Triangle Congruence
================================================================================

  Statement: SSS: side-side-side triangle congruence.

  Given:     ab = de, bc = ef, ca = fd
  Prove:     ∠bac = ∠edf, ∠abc = ∠def, ∠bca = ∠efd, △abc = △def

  Dependencies: I.7 (superposition axiom)

  Proof (3 steps):
  ─────────────────
  1. [SUPERPOSITION_SSS] SSS superposition
     var_map: {a→a, b→b, c→c, d→d, e→e, f→f}
     → ∠bac = ∠edf, ∠abc = ∠def, ∠acb = ∠dfe

  2. [METRIC] ∠bca = ∠efd (M4 symmetry)
     → ∠bca = ∠efd

  3. [METRIC] △abc = △def (M9)
     → △abc = △def

  Q.E.D. ■


================================================================================
  PROPOSITION I.9 — Bisect an Angle
================================================================================

  Statement: To bisect a given rectilineal angle.

  Given:     ¬(a = b), ¬(a = c), ¬(b = c), on(a, M), on(b, M),
             on(a, N), on(c, N), same-side(c, b, M), same-side(b, c, N)
  Construct: ∃e : POINT
  Prove:     ∠bae = ∠cae, same-side(e, c, M), same-side(e, b, N)

  Dependencies: I.1, I.3, I.8

  Proof (5 steps):
  ─────────────────
  1. [CONSTRUCTION] let-point-on-line
     → new: d (point)
     → on(d, M), ¬(d = a)

  2. [CONSTRUCTION] Cut equal segment on ray ac (I.3)
     → new: f (point)
     → on(f, N), ad = af

  3. [CONSTRUCTION] Equilateral triangle on df (I.1)
     → new: e (point)
     → df = de, df = fe

  4. [METRIC] By SSS (I.8): ∠bae = ∠cae
     → ∠bae = ∠cae

  5. [DIAGRAMMATIC] same-side conclusions
     → same-side(e, c, M), same-side(e, b, N)

  Q.E.F. ■


================================================================================
  PROPOSITION I.10 — Bisect a Segment
================================================================================

  Statement: To bisect a given finite straight line.

  Given:     on(a, L), on(b, L), ¬(a = b)
  Construct: ∃d : POINT
  Prove:     between(a, d, b), ad = db

  Dependencies: I.1, I.4

  Proof (2 steps):
  ─────────────────
  1. [THEOREM_APP] Equilateral triangle on ab (I.1)
     var_map: {a→a, b→b, c→c}
     → new: c (point)
     → ab = ac, ab = bc, ¬(c = a), ¬(c = b)

  2. [CONSTRUCTION] Bisect angle acb (I.9) → d on L between a and b
     → new: d (point)
     → between(a, d, b), ad = db

  Q.E.F. ■


================================================================================
  PROPOSITION I.11 — Perpendicular from a Point on a Line
================================================================================

  Statement: To draw a straight line at right angles to a given straight
             line from a given point on it.

  Given:     on(a, L), on(b, L), ¬(a = b)
  Construct: ∃f : POINT
  Prove:     ∠baf = R, ¬(f = a), ¬on(f, L)

  Dependencies: I.1, I.3, I.8

  Proof (4 steps):
  ─────────────────
  1. [CONSTRUCTION] Symmetric point d on L with ad = ab (I.3)
     → new: d (point)
     → on(d, L), ¬(d = a), ad = ab

  2. [CONSTRUCTION] Equilateral triangle on db (I.1)
     → new: f (point)
     → db = df, db = bf

  3. [DIAGRAMMATIC] f is not on L
     → ¬on(f, L), ¬(f = a)

  4. [METRIC] By SSS (I.8): ∠baf = right angle
     → ∠baf = R

  Q.E.F. ■


================================================================================
  PROPOSITION I.12 — Perpendicular from a Point Off a Line
================================================================================

  Statement: To draw a straight line perpendicular to a given infinite
             straight line from a given point not on it.

  Given:     on(a, L), on(b, L), ¬(a = b), ¬on(p, L)
  Construct: ∃h : POINT
  Prove:     on(h, L), ∠ahp = R, ¬(h = p)

  Dependencies: I.8, I.10

  Proof (1 step):
  ────────────────
  1. [CONSTRUCTION] Drop perpendicular via circle + bisect (I.8, I.10)
     → new: h (point)
     → on(h, L), ∠ahp = R, ¬(h = p)

  Q.E.F. ■


================================================================================
  PROPOSITION I.13 — Supplementary Angles
================================================================================

  Statement: If a straight line set up on a straight line makes angles,
             it makes either two right angles or angles equal to two
             right angles.

  Given:     on(a, L), on(c, L), between(a, b, c), ¬on(d, L), ¬(b = d)
  Prove:     ∠abd + ∠dbc = R + R

  Dependencies: I.11

  Proof (2 steps):
  ─────────────────
  1. [CONSTRUCTION] Draw perpendicular at b (I.11)
     → new: e (point)
     → ∠abe = R, ¬on(e, L)

  2. [METRIC] ∠abd + ∠dbc = 2R
     → ∠abd + ∠dbc = R + R

  Q.E.D. ■


================================================================================
  PROPOSITION I.14 — Converse of I.13
================================================================================

  Statement: If with any straight line, and at a point on it, two straight
             lines not lying on the same side make the adjacent angles
             equal to two right angles, the two lines are in a straight
             line with one another.

  Given:     on(a, L), on(b, L), ¬(a = b), ¬on(c, L), ¬on(d, L),
             ¬(b = c), ¬(b = d), ¬same-side(c, d, L),
             ∠abc + ∠abd = R + R
  Prove:     between(c, b, d)

  Dependencies: I.13

  Proof (1 step):
  ────────────────
  1. [METRIC] By I.13: angles sum forces collinearity
     → between(c, b, d)

  Q.E.D. ■


================================================================================
  PROPOSITION I.15 — Vertical Angles
================================================================================

  Statement: If two straight lines cut one another, they make the
             vertical angles equal to one another.

  Given:     on(a, L), on(b, L), on(c, M), on(d, M), on(e, L), on(e, M),
             between(a, e, b), between(c, e, d), ¬(L = M)
  Prove:     ∠aec = ∠bed

  Dependencies: I.13

  Proof (3 steps):
  ─────────────────
  1. [METRIC] ∠aec + ∠ceb = 2R (I.13)
     → ∠aec + ∠ceb = R + R

  2. [METRIC] ∠ced + ∠deb = 2R (I.13)
     → ∠ced + ∠deb = R + R

  3. [METRIC] Therefore ∠aec = ∠bed (common supplement)
     → ∠aec = ∠bed

  Q.E.D. ■


================================================================================
  PROPOSITION I.16 — Exterior Angle Theorem
================================================================================

  Statement: In any triangle, if one of the sides is produced, the
             exterior angle is greater than either of the interior and
             opposite angles.

  Given:     on(a, L), on(b, L), between(a, b, d), ¬on(c, L),
             ¬(a = b), ¬(a = c), ¬(b = c)
  Prove:     ∠bac < ∠dbc, ∠bca < ∠dbc

  Dependencies: I.3, I.4, I.10, I.15

  Proof (6 steps):
  ─────────────────
  1. [CONSTRUCTION] Bisect bc at e (I.10)
     → new: e (point)
     → between(b, e, c), be = ec

  2. [CONSTRUCTION] Extend ae to f with ae = ef (I.3)
     → new: f (point)
     → between(a, e, f), ae = ef

  3. [CONSTRUCTION] let-line
     → new: P (line)
     → on(b, P), on(f, P)

  4. [DIAGRAMMATIC] distinctness
     → ¬(e = f), ¬(e = a)

  5. [METRIC] By SAS (I.4): △abe ≅ △cef → ∠bae = ∠fce
     → ∠bae = ∠fce

  6. [METRIC] ∠bac < ∠dbc and ∠bca < ∠dbc
     → ∠bac < ∠dbc, ∠bca < ∠dbc

  Q.E.D. ■


================================================================================
  PROPOSITION I.17 — Two Angles Less Than Two Right Angles
================================================================================

  Statement: In any triangle two angles taken together in any manner are
             less than two right angles.

  Given:     ¬(a = b), ¬(a = c), ¬(b = c), on(a, L), on(b, L), ¬on(c, L)
  Prove:     ∠abc + ∠bca < R + R

  Dependencies: I.13, I.16

  Proof (1 step):
  ────────────────
  1. [METRIC] Extend bc, exterior angle (I.16) + supplementary (I.13)
     → ∠abc + ∠bca < R + R

  Q.E.D. ■


================================================================================
  PROPOSITION I.18 — Greater Side Subtends Greater Angle
================================================================================

  Statement: In any triangle the greater side subtends the greater angle.

  Given:     ¬(a = b), ¬(a = c), ¬(b = c), ab < ac
  Prove:     ∠abc < ∠acb

  Dependencies: I.3, I.5, I.16

  Proof (1 step):
  ────────────────
  1. [METRIC] Cut equal segment (I.3), isosceles (I.5), exterior (I.16)
     → ∠abc < ∠acb

  Q.E.D. ■


================================================================================
  PROPOSITION I.19 — Greater Angle Subtended by Greater Side
================================================================================

  Statement: In any triangle the greater angle is subtended by the
             greater side.

  Given:     ¬(a = b), ¬(a = c), ¬(b = c), ∠abc < ∠acb
  Prove:     ab < ac

  Dependencies: I.5, I.18

  Proof (1 step):
  ────────────────
  1. [METRIC] Contrapositive of I.5 + I.18
     → ab < ac

  Q.E.D. ■


================================================================================
  PROPOSITION I.20 — Triangle Inequality
================================================================================

  Statement: In any triangle two sides taken together in any manner are
             greater than the remaining one.

  Given:     ¬(a = b), ¬(a = c), ¬(b = c)
  Prove:     bc < ab + ac

  Dependencies: I.3, I.5, I.19

  Proof (4 steps):
  ─────────────────
  1. [CONSTRUCTION] Extend ba to d with ad = ac (I.3)
     → new: d (point)
     → between(b, a, d), ad = ac

  2. [CONSTRUCTION] let-line
     → new: M (line)
     → on(d, M), on(c, M)

  3. [METRIC] ∠acd = ∠adc (isosceles, I.5)
     → ∠acd = ∠adc

  4. [METRIC] ∠bcd > ∠bdc → bc < bd = ba + ac (I.19)
     → bc < ab + ac

  Q.E.D. ■


================================================================================
  PROPOSITION I.21 — Inner Triangle Inequality
================================================================================

  Statement: If on one of the sides of a triangle, from its extremities,
             there be constructed two straight lines meeting within the
             triangle, they will be less than the remaining two sides but
             will contain a greater angle.

  Given:     ¬(a = b), ¬(a = c), ¬(b = c), ¬(d = b), ¬(d = c),
             on(b, L), on(c, L), ¬on(a, L), same-side(d, a, L)
  Prove:     bd + dc < ba + ac, ∠bac < ∠bdc

  Dependencies: I.16, I.20

  Proof (1 step):
  ────────────────
  1. [METRIC] Extend bd to meet ac; I.16 gives angle, I.20 gives sides
     → bd + dc < ba + ac, ∠bac < ∠bdc

  Q.E.D. ■

  PROPOSITION I.22 — Construct Triangle from Three Segments
================================================================================

  Statement: Out of three straight lines, which are equal to three given
             straight lines, to construct a triangle.

  Given:     ¬(a = b), ¬(c = d), ¬(e = f),
             ab < cd + ef, cd < ab + ef, ef < ab + cd
  Construct: ∃p, q, r : POINT
  Prove:     pq = ab, pr = cd, qr = ef

  Dependencies: I.1, I.3, I.20

  Proof (1 step):
  ────────────────
  1. [CONSTRUCTION] Construct triangle from three segments (I.3, I.1)
     → new: p, q, r (points)
     → pq = ab, pr = cd, qr = ef

  Q.E.F. ■

  PROPOSITION I.23 — Copy an Angle
================================================================================

  Statement: On a given straight line and at a point on it, to construct
             a rectilineal angle equal to a given rectilineal angle.

  Given:     ¬(d = e), ¬(d = f), ¬(e = f), on(a, L), on(b, L), ¬(a = b)
  Construct: ∃g : POINT
  Prove:     ∠bag = ∠edf, ¬on(g, L)

  Dependencies: I.8, I.22

  Proof (1 step):
  ────────────────
  1. [CONSTRUCTION] Triangle from sides of angle (I.22), then SSS (I.8)
     → new: g (point)
     → ∠bag = ∠edf, ¬on(g, L)

  Q.E.F. ■


================================================================================
  PROPOSITION I.24 — SAS Inequality (Hinge Theorem)
================================================================================

  Statement: If two triangles have two sides equal respectively but the
             included angle of the first greater than that of the second,
             the base of the first is also greater.

  Given:     ab = de, ac = df, ∠edf < ∠bac,
             ¬(a = b), ¬(a = c), ¬(b = c), ¬(d = e), ¬(d = f), ¬(e = f)
  Prove:     ef < bc

  Dependencies: I.3, I.4, I.5, I.19, I.23

  Proof (1 step):
  ────────────────
  1. [METRIC] Copy angle (I.23), SAS (I.4), isosceles (I.5), I.19
     → ef < bc

  Q.E.D. ■


================================================================================
  PROPOSITION I.25 — Converse Hinge Theorem
================================================================================

  Statement: If two triangles have two sides equal respectively but the
             base of the first greater, the angle contained by the equal
             sides of the first is also greater.

  Given:     ab = de, ac = df, ef < bc,
             ¬(a = b), ¬(a = c), ¬(b = c), ¬(d = e), ¬(d = f), ¬(e = f)
  Prove:     ∠edf < ∠bac

  Dependencies: I.4, I.24

  Proof (1 step):
  ────────────────
  1. [METRIC] Contrapositive of I.24 + SAS (I.4)
     → ∠edf < ∠bac

  Q.E.D. ■


================================================================================
  PROPOSITION I.26 — ASA / AAS Congruence
================================================================================

  Statement: If two triangles have the two angles equal to two angles
             respectively, and one side equal to one side, they will also
             have the remaining sides equal and the remaining angle equal.

  Given:     ∠abc = ∠def, ∠bca = ∠efd, bc = ef,
             ¬(a = b), ¬(a = c), ¬(b = c), ¬(d = e), ¬(d = f), ¬(e = f)
  Prove:     ab = de, ac = df, ∠bac = ∠edf, △abc = △def

  Dependencies: I.3, I.4, I.16

  Proof (2 steps):
  ─────────────────
  1. [CONSTRUCTION] Suppose ab ≠ de, cut g with dg = ab (I.3)
     → new: g (point)
     → dg = ab

  2. [METRIC] SAS (I.4) → contradiction → full congruence
     → ab = de, ac = df, ∠bac = ∠edf, △abc = △def

  Q.E.D. ■


================================================================================
  PROPOSITION I.27 — Alternate Interior Angles ⇒ Parallel
================================================================================

  Statement: If a straight line falling on two straight lines makes the
             alternate angles equal, the straight lines are parallel.

  Given:     on(a, L), on(b, L), on(b, M), on(c, M), on(c, N), on(d, N),
             ¬(a = b), ¬(b = c), ¬(c = d), ¬(L = N),
             ¬same-side(a, d, M), ∠abc = ∠bcd
  Prove:     ¬intersects(L, N)

  Dependencies: I.16

  Note: This is the last major proposition proved without the parallel
        postulate (neutral geometry).

  Proof (1 step):
  ────────────────
  1. [METRIC] If not parallel, intersection → exterior angle
     contradiction (I.16)
     → ¬intersects(L, N)

  Q.E.D. ■


================================================================================
  PROPOSITION I.28 — Co-interior Angles ⇒ Parallel
================================================================================

  Statement: If a straight line falling on two straight lines makes the
             exterior angle equal to the interior and opposite angle on
             the same side, or the co-interior angles sum to two right
             angles, the straight lines are parallel.

  Given:     on(a, L), on(b, L), on(b, M), on(c, M), on(c, N), on(d, N),
             ¬(a = b), ¬(b = c), ¬(c = d), ¬(L = N),
             same-side(a, d, M), ∠abc + ∠bcd = R + R
  Prove:     ¬intersects(L, N)

  Dependencies: I.13, I.15, I.27

  Proof (1 step):
  ────────────────
  1. [METRIC] Equal exterior angles → alternate angles equal → I.27
     → ¬intersects(L, N)

  Q.E.D. ■


================================================================================
  PROPOSITION I.29 — Parallel ⇒ Alternate Interior Angles Equal
================================================================================

  Statement: A straight line falling on parallel straight lines makes the
             alternate angles equal, the exterior angle equal to the
             interior and opposite, and the co-interior angles equal to
             two right angles.

  Given:     on(a, L), on(b, L), on(b, M), on(c, M), on(c, N), on(d, N),
             ¬(a = b), ¬(b = c), ¬(c = d), ¬(L = N),
             ¬same-side(a, d, M), ¬intersects(L, N)
  Prove:     ∠abc = ∠bcd

  Dependencies: I.27, Postulate 5 (parallel postulate)

  Note: FIRST use of the parallel postulate. Everything before I.29 is
        neutral geometry.

  Proof (1 step):
  ────────────────
  1. [METRIC] Parallel lines: alternate angles equal (Playfair via I.27)
     → ∠abc = ∠bcd

  Q.E.D. ■


================================================================================
  PROPOSITION I.30 — Transitivity of Parallelism
================================================================================

  Statement: Straight lines parallel to the same straight line are also
             parallel to one another.

  Given:     ¬intersects(L, M), ¬intersects(M, N),
             ¬(L = M), ¬(M = N), ¬(L = N)
  Prove:     ¬intersects(L, N)

  Dependencies: I.27, I.29

  Proof (1 step):
  ────────────────
  1. [METRIC] Transitive parallelism via I.27 + I.29
     → ¬intersects(L, N)

  Q.E.D. ■


================================================================================
  PROPOSITION I.31 — Construct a Parallel Through a Point
================================================================================

  Statement: Through a given point to draw a straight line parallel to a
             given straight line.

  Given:     on(b, L), on(c, L), ¬(b = c), ¬on(a, L)
  Construct: ∃M : LINE
  Prove:     on(a, M), ¬intersects(L, M)

  Dependencies: I.23, I.27

  Proof (1 step):
  ────────────────
  1. [CONSTRUCTION] Copy angle (I.23) → parallel (I.27)
     → new: e (point)
     → on(a, M), ¬intersects(L, M)

  Q.E.F. ■


================================================================================
  PROPOSITION I.32 — Angle Sum of a Triangle
================================================================================

  Statement: In any triangle, if one of the sides is produced, the exterior
             angle equals the two remote interior angles, and the three
             interior angles equal two right angles.

  Given:     on(a, L), ¬(a = b), ¬(a = c), ¬(b = c),
             between(b, c, d), ¬(c = d)
  Prove:     ∠acd = ∠cab + ∠abc,
             ∠abc + ∠bca + ∠cab = R + R

  Dependencies: I.13, I.29, I.31

  Proof (2 steps):
  ─────────────────
  1. [CONSTRUCTION] Draw parallel through c (I.31)
     → new: e (point)
     → ¬(e = c)

  2. [METRIC] Alternate angles (I.29) + supplementary (I.13) → conclusions
     → ∠acd = ∠cab + ∠abc,
       ∠abc + ∠bca + ∠cab = R + R

  Q.E.D. ■


================================================================================
  PROPOSITION I.33 — Joining Equal Parallel Segments
================================================================================

  Statement: The straight lines joining equal and parallel straight lines
             at the extremities which are in the same directions
             respectively are themselves also equal and parallel.

  Given:     on(a, L), on(b, L), on(c, N), on(d, N),
             ¬intersects(L, N), ¬(a = b), ¬(c = d), ab = cd,
             on(a, M), on(c, M), ¬(L = N)
  Prove:     ac = bd, ¬intersects(M, L)

  Dependencies: I.4, I.27, I.29

  Proof (1 step):
  ────────────────
  1. [METRIC] Join diagonal, I.4 + I.27 → parallelogram
     → ac = bd, ¬intersects(M, L)

  Q.E.D. ■


================================================================================
  PROPOSITION I.34 — Properties of a Parallelogram
================================================================================

  Statement: In a parallelogram the opposite sides and angles are equal
             to one another, and the diagonal bisects the area.

  Given:     on(a, L), on(b, L), on(c, N), on(d, N), ¬intersects(L, N),
             on(a, M), on(d, M), on(b, P), on(c, P), ¬intersects(M, P),
             ¬(a = b), ¬(c = d), ¬(a = d), ¬(b = c)
  Prove:     ab = cd, ad = bc, ∠dab = ∠bcd, ∠abc = ∠cda, △abc = △acd

  Dependencies: I.4, I.26, I.29

  Proof (1 step):
  ────────────────
  1. [METRIC] Diagonal + I.29 alternate angles + ASA (I.26) → all
     → ab = cd, ad = bc, ∠dab = ∠bcd, ∠abc = ∠cda, △abc = △acd

  Q.E.D. ■


================================================================================
  PROPOSITION I.35 — Parallelograms on Same Base, Same Parallels
================================================================================

  Statement: Parallelograms which are on the same base and in the same
             parallels are equal to one another.

  Given:     on(b, N), on(c, N), on(a, L), on(d, L), on(e, L), on(f, L),
             ¬intersects(L, N), ¬(b = c), ¬(a = d), ¬(e = f)
  Prove:     △abc + △acd = △ebc + △ecf

  Dependencies: I.29, I.34

  Proof (1 step):
  ────────────────
  1. [METRIC] Same base + parallels: I.29 + I.34 → equal areas
     → △abc + △acd = △ebc + △ecf

  Q.E.D. ■


================================================================================
  PROPOSITION I.36 — Parallelograms on Equal Bases, Same Parallels

  Statement: Parallelograms which are on equal bases and in the same
             parallels are equal to one another.

  Given:     on(b, N), on(c, N), on(e, N), on(f, N), on(a, L), on(d, L),
             ¬intersects(L, N), bc = ef, ¬(b = c), ¬(e = f)

  Dependencies: I.33, I.34, I.35

  Q.E.D. ■

  PROPOSITION I.37 — Triangles on Same Base, Same Parallels

  PROPOSITION I.39 — Equal Triangles on Same Base ⇒ Same Parallels

  Statement: Equal triangles which are on the same base and on the same

  PROPOSITION I.41 — Parallelogram Double the Triangle
================================================================================

  Statement: If a parallelogram have the same base with a triangle and be
             in the same parallels, the parallelogram is double of the
             triangle.

  Given:     on(b, N), on(c, N), on(a, L), on(d, L), ¬intersects(L, N),
             ¬(b = c), ¬(a = d), on(e, L), ¬on(e, N)
  Prove:     △abc + △acd = △ebc + △ebc

  Dependencies: I.34, I.37

  Proof (1 step):
  ────────────────
  1. [METRIC] Parallelogram = 2 × triangle (I.34 diagonal + I.37)
     → △abc + △acd = △ebc + △ebc

  Q.E.D. ■

  PROPOSITION I.47 — The Pythagorean Theorem
================================================================================

  Statement: In right-angled triangles the square on the side subtending
             the right angle is equal to the squares on the sides
             containing the right angle.

  Given:     ¬(a = b), ¬(a = c), ¬(b = c), ∠bac = R
  Prove:     bc² = ab² + ac²
             [expressed as: 2·△(b,c,b) = 2·△(a,b,a) + 2·△(a,c,a)]

  Dependencies: I.4, I.14, I.41, I.46

  Proof (1 step):
  ────────────────
  1. [METRIC] Squares on sides (I.46), parallels (I.31),
     2×triangle = rect (I.41), I.4 → Pythagorean
     → bc² = ab² + ac²

  Q.E.D. ■


================================================================================
  PROPOSITION I.48 — Converse of the Pythagorean Theorem
================================================================================

  Statement: If in a triangle the square on one of the sides be equal to
             the squares on the remaining two sides, the angle contained
             by the remaining two sides is right.

  Given:     ¬(a = b), ¬(a = c), ¬(b = c), bc² = ab² + ac²
  Prove:     ∠bac = R

  Dependencies: I.3, I.8, I.11, I.46, I.47

  Proof (1 step):
  ────────────────
  1. [METRIC] Construct right triangle (I.11), apply I.47, SSS (I.8)
     → ∠bac = R

  Q.E.D. ■
